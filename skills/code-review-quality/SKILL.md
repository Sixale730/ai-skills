# Code Review Quality

Prioritize feedback: Blocker (must fix) > Major > Minor > Suggestion.
Focus on: Bugs, security, testability, maintainability (not style preferences).
Ask questions over commands. Provide context. Limit scope to < 400 lines.

Feedback levels: CRITICAL, HIGH, MEDIUM, LOW, INFORMATIONAL.
Minimum weighted finding score of 3.0 per review.
